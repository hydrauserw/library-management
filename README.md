# 📚 Библиотека «Читай-Город»
- Система учёта книг и выдачи читателям  
- Языки и методы программирования

### Возможности
- Каталог книг с остатками в реальном времени
- Регистрация читателей и лимит выдачи (3 книги)
- Выдача/возврат с автоматическим обновлением остатков
- Отслеживание сроков возврата (14 дней)
- Статистика: активные выдачи, просрочки, популярность
- Сохранение данных в JSON-файлы
